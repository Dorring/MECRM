# Evidence Capture Checklist

Capture only after the local prerequisites in the
[evidence capture map](evidence-capture-map.md) are satisfied.

- [ ] Support Copilot structured result with tenant-safe evidence references.
- [ ] Pending approval state.
- [ ] Approved or rejected result with audit record.
- [ ] Agent-run trace with redacted tool/evidence summaries.
- [ ] Prompt-injection or cross-tenant denial state.
- [ ] AI evaluation report with commit and dataset version.
- [ ] Grafana or metrics view showing agent operational evidence.

Before publishing any image or video, remove URLs, passwords, access tokens,
email addresses, customer data, and internal hostnames.
