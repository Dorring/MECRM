# Chaos Engineering Proof Artifacts

These files are generated by the isolated chaos test job and local chaos runs.

## Generated JSON reports

- `kafka_recovery.json`
  - proves Kafka outage recovery without outbox message loss
- `consumer_recovery.json`
  - proves consumer crash recovery with idempotent projections
- `db_recovery.json`
  - proves DB transient failure recovery with bounded retries and circuit breaking

## How these are produced

- Local: run the chaos suite with `CHAOS_TESTS_ENABLED=true` and `CHAOS_ENVIRONMENT=local`
- CI: `.github/workflows/chaos-tests.yml` runs nightly or manually and uploads this folder as an artifact
